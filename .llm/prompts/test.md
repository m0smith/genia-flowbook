You are working on this project.

Before making any changes:
- Read `AGENTS.md`, `PROJECT_STATE.md`, `PROJECT_RULES.md`, and `README.md`.
- Read the approved spec, approved design, and implemented code.
- Treat `PROJECT_STATE.md` as final authority.

## Task type: TESTS ONLY

Add or update tests for the approved and implemented feature.

Do NOT:
- implement the feature
- redesign the feature
- broaden scope
- weaken existing tests to make behavior pass

- edge cases
- failure cases
- regression protection outside the approved scope

## Rules
- prefer black-box tests where practical
- keep tests focused
- assert real behavior, not vague success
- preserve truthfulness
